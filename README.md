# Binance Futures Testnet Trading Bot

A Python CLI application that places market and limit orders on Binance Futures Testnet (USDT-M).  
Built as part of the Python Developer Intern assignment for Anything.ai.

## ✅ Core Requirements Met

- [x] Place **Market** and **Limit** orders (BUY/SELL)
- [x] CLI with argparse (validates symbol, side, type, quantity, price)
- [x] Structured code: separate client, orders, validators, logging layers
- [x] Logging to file (`trading_bot.log`) with API requests, responses, errors
- [x] Comprehensive exception handling (invalid input, API errors, network)
- [x] Clear output: order summary + response details
